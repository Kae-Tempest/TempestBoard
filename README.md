## Installation

### After Cloning repository

#### .ENV

Modify example.env to .env and modify .env content if you need

#### Linux / Mac OS

```bash
cd TempestBoard
sudo docker compose up -d
```

#### Windows

```bash
cd TempestBoard
docker-compose up -d
```
